# YAN Portfolio

YAN Portfolio là ứng dụng portfolio cá nhân của Nguyễn Đặng Trường An (Yami An), xây bằng React và Create React App. Source tập trung vào cấu hình nội dung trong `src/portfolio.js`, render các trang Home, Education, Experience, Projects, Open Source và Contact bằng component tái sử dụng, theme tùy biến và dữ liệu snapshot JSON.

## Stack kỹ thuật

- React 16, React DOM và `react-router-dom` với `HashRouter`.
- Create React App (`react-scripts`) cho dev server, test và production build.
- `styled-components`, `styletron-react`, `styletron-engine-atomic` và BaseUI cho theme/provider.
- `react-reveal` cho animation section/card.
- `react-bootstrap` cho tooltip/overlay.
- `chart.js` và `react-chartjs-2` cho biểu đồ đóng góp open-source.
- Font Awesome vendored trong `src/assests/font-awesome` và icon/ảnh kỹ năng trong `public/skills`.

## Kiến trúc source

```text
src/
  App.js                       # Root app, gắn ThemeProvider và router chính
  containers/Main.js           # Khai báo route HashRouter và 404 fallback
  portfolio.js                 # Nguồn dữ liệu nội dung chính của portfolio
  theme.js                     # Bảng theme màu và chosenTheme
  components/                  # Button, header, card, chart, social, footer...
  containers/                  # Section tái sử dụng cho home/open-source/legacy
  pages/                       # Màn hình route-level
  shared/opensource/*.json     # Snapshot project, PR, issue, organization
  assests/images/              # Ảnh và SVG minh họa
```

`src/assests/font-awesome` là vendor/autogenerated asset nên không được chỉnh tay hoặc thêm comment nội bộ. Các file source tự viết đã có summary/comment tiếng Việt ở đầu file; CSS/SCSS tự viết cũng có header mô tả vai trò stylesheet.

## Luồng chạy chính

1. `src/index.js` tạo Styletron engine, gắn BaseUI provider và render `App`.
2. `App` gắn `ThemeProvider`, `GlobalStyles` và truyền `chosenTheme` xuống `Main`.
3. `Main` chọn landing page theo `settings.isSplash` trong `portfolio.js`, sau đó khai báo route cho Home, Education, Experience, Projects, Open Source, Contact và 404.
4. Các page lấy nội dung từ `portfolio.js` hoặc các snapshot trong `src/shared/opensource`.
5. `SeoHeader` sinh meta tag và JSON-LD Person từ dữ liệu portfolio.

## Cấu hình nội dung

- Thông tin cá nhân, social links, SEO, kỹ năng, chứng chỉ, kinh nghiệm, project header và contact nằm trong `src/portfolio.js`.
- Danh sách project hiển thị trên trang Projects lấy từ `src/shared/opensource/projects.json`.
- Trang Open Source dùng `organizations.json`, `pull_requests.json` và `issues.json` để render logo tổ chức, chart và card đóng góp.
- Theme hiện tại là `blueTheme`, được export qua `chosenTheme` trong `src/theme.js`.
- `settings.isSplash = true` sẽ bật splash route trước khi chuyển sang `/home`.

## Lệnh phát triển

```bash
npm install
npm start
npm run lint
npm test -- --watchAll=false
npm run build
```

`npm run build` tạo thư mục `build/` để deploy tĩnh. Thư mục này đã được ignore trong Git vì là artifact sinh ra từ CRA.

## CI, Docker và deploy

- `Dockerfile` dùng `node:22-alpine`, cài dependency bằng `npm ci` từ lockfile và chạy dev server bằng `npm start` trên port `3000`.
- `Jenkinsfile` mô tả pipeline CI/CD có bước checkout, install, build Docker image, push registry và deploy.
- Script `deploy` dùng `gh-pages -b gh-pages -d build` nếu muốn publish build tĩnh lên branch `gh-pages`.

## Ghi chú bảo trì

- ESLint bỏ qua `src/assests/font-awesome` vì đây là vendor/minified bundle.
- Dependency runtime đã được dọn để loại Apollo GraphQL/fetch/radium/react-spring không còn dùng.
- `npm audit fix` thường đã được chạy; một số advisory còn lại nằm trong chuỗi `react-scripts`/Jest/Webpack cũ và npm chỉ đề xuất `--force` breaking change, nên chưa ép nâng.
- Test hiện pass nhưng có warning từ `react-reveal` do thư viện dùng lifecycle cũ của React.
